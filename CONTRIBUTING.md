# Contributing Guidelines

## Getting Started

1. Fork the repository.
2. Clone your fork.
3. Install the required dependencies.

## Reporting Bugs

Please use GitHub Issues to report any bugs, providing detailed information and reproduction steps.

## Adding New Visualizations

Follow the guidelines in [EXTENDVISUALIZATIONS.md](EXTENDVISUALIZATIONS.md) for adding new visualizations. Ensure your code passes the required tests.

## Coding Standards

We follow the [Airbnb JavaScript Style Guide](https://github.com/airbnb/javascript).

## Pull Requests

1. Create a new branch for your feature/bugfix.
2. Commit your changes.
3. Push to your fork and submit a pull request.

## Testing

Before submitting your pull request, run the tests with:

```bash
npm test
```

Ensure all tests are passing.
